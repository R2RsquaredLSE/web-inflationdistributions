# Data

Drop the Stata-format probability estimates here:

| Filename | Description |
|---|---|
| `USwestimates.dta` | US inflation disaster probabilities (time series) |
| `EZwestimates.dta` | Euro Area inflation disaster probabilities (time series) |

The variables exposed by each `.dta` are documented in the top-level `README.md`.

Once the files are in place, the download links at the root of the project page
(`data/USwestimates.dta`, `data/EZwestimates.dta`) will resolve correctly.
